# Dataset

## Description

This folder contains the dataset generated from the Aspen Dynamics simulation of the Ethyl Propionate reactive distillation process.

The dataset was used for training, validating, and evaluating deep learning-based soft sensor models.

## Dataset Information

- Source: Aspen Dynamics V14
- Simulation Type: Dynamic Simulation
- Number of Input Features: 36
- Number of Output Variables: 2

## Target Variables

- Ethyl Propionate Mole Fraction
- Ethanol Mole Fraction
