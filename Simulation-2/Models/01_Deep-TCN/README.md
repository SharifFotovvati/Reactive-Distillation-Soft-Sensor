# Deep Temporal Convolutional Network (Deep TCN)

## Overview

This folder contains the implementation of the Deep Temporal Convolutional Network (Deep TCN) developed for predicting key process variables in the Ethyl Propionate reactive distillation process.

## Model Information

- Architecture: Deep Temporal Convolutional Network
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Ethyl Propionate Mole Fraction
- Ethanol Mole Fraction

## Files

- DeepTCN_EthylPropionate_Soft_Sensor.ipynb
- Architecture.png (Optional)

The Deep TCN model captures long-range temporal dependencies using dilated causal convolutions, providing accurate prediction of nonlinear process dynamics.
