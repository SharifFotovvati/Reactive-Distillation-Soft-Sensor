# LSTM with Attention

## Overview

This folder contains the implementation of the LSTM with Attention model developed for predicting key process variables in the Ethyl Propionate reactive distillation process.

## Model Information

- Architecture: Long Short-Term Memory (LSTM) with Attention Mechanism
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Ethyl Propionate Mole Fraction
- Ethanol Mole Fraction

## Files

- LSTM_Attention_EthylPropionate_Soft_Sensor.ipynb
- Architecture.png (Optional)

The LSTM with Attention model combines the sequence learning capability of Long Short-Term Memory networks with an attention mechanism to selectively focus on the most informative process variables, improving prediction accuracy for nonlinear dynamic systems.
