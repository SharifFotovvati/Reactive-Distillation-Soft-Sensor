# Value Attention LSTM (VA-LSTM)

## Overview

This folder contains the implementation of the Value Attention Long Short-Term Memory (VA-LSTM) model developed for predicting key process variables in the Ethyl Propionate reactive distillation process.

## Model Information

- Architecture: Value Attention Long Short-Term Memory (VA-LSTM)
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Ethyl Propionate Mole Fraction
- Ethanol Mole Fraction

## Files

- VALSTM_EthylPropionate_Soft_Sensor.ipynb
- Architecture.png (Optional)

The VA-LSTM model incorporates a value attention mechanism to dynamically emphasize the most informative process variables during prediction, improving the accuracy and robustness of soft sensor modeling for reactive distillation systems.
