# Temporal Convolutional Network (TCN)

## Overview

This folder contains the implementation of the Temporal Convolutional Network (TCN) developed for predicting key process variables in the MTBE reactive distillation process.

## Model Information

- Architecture: Temporal Convolutional Network (TCN)
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Methanol Mole Fraction
- 1-Butene Mole Fraction

## Files

- TCN_MTBE_Soft_Sensor.ipynb
- Architecture.png (Optional)

This model employs temporal convolutional layers to capture long-range dependencies in process data for soft sensor applications.
