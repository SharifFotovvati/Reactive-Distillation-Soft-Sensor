# Variable Attention LSTM (VA-LSTM)

## Overview

This folder contains the implementation of the Variable Attention Long Short-Term Memory (VA-LSTM) model developed for predicting key process variables in the MTBE reactive distillation process.

## Model Information

- Architecture: Variable Attention Long Short-Term Memory (VA-LSTM)
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Methanol Mole Fraction
- 1-Butene Mole Fraction

## Files

- VALSTM_MTBE_Soft_Sensor.ipynb
- Architecture.png (Optional)

The VA-LSTM model employs a variable attention mechanism to dynamically assign importance to different process variables, improving prediction accuracy for nonlinear reactive distillation systems.
