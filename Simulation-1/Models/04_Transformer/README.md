# Transformer

## Overview

This folder contains the implementation of the Transformer model developed for predicting key process variables in the MTBE reactive distillation process.

## Model Information

- Architecture: Transformer
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Methanol Mole Fraction
- 1-Butene Mole Fraction

## Files

- Transformer_MTBE_Soft_Sensor.ipynb
- Architecture.png (Optional)

The Transformer model utilizes the self-attention mechanism to capture complex relationships among process variables, enabling accurate soft sensor prediction for reactive distillation systems.
