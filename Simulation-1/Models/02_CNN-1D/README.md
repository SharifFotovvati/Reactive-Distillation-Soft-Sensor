# One-Dimensional Convolutional Neural Network (CNN-1D)

## Overview

This folder contains the implementation of the One-Dimensional Convolutional Neural Network (CNN-1D) developed for predicting key process variables in the MTBE reactive distillation process.

## Model Information

- Architecture: One-Dimensional Convolutional Neural Network
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Methanol Mole Fraction
- 1-Butene Mole Fraction
