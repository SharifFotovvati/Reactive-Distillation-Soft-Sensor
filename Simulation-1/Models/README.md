This folder contains all deep learning models developed for the MTBE reactive distillation soft sensor.

## Implemented Models

1. Feedforward Neural Network (FNN)
2. One-Dimensional Convolutional Neural Network (CNN-1D)
3. Temporal Convolutional Network (TCN)
4. Transformer
5. Gated Stacked Autoencoder (GSAE)
6. Variable Attention LSTM (VA-LSTM)
7. Self-Attention LSTM (SA-LSTM)

Each model includes:

- Source code
- Model architecture
- Input and output description
- Training configuration
- Performance evaluation
