# Gated Stacked Autoencoder (GSAE)

## Overview

This folder contains the implementation of the Gated Stacked Autoencoder (GSAE) developed for soft sensor modeling of the MTBE reactive distillation process.

## Model Information

- Architecture: Gated Stacked Autoencoder (GSAE)
- Framework: TensorFlow / Keras
- Input Features: 36
- Output Variables: 2

### Outputs

- Methanol Mole Fraction
- 1-Butene Mole Fraction

## Files

- GSAE_MTBE_Soft_Sensor.ipynb
- Architecture.png (Optional)

The GSAE model combines stacked autoencoder layers with gating mechanisms to improve nonlinear feature extraction and prediction accuracy for reactive distillation soft sensing.
